## About This Course

- A **1-day intensive** hands-on training focused on **Agentic AI** for firmware and embedded systems development.
- The focus is on **Autonomous Agents** that understand registers, timing constraints, and power states - not just general coding assistants.

## Pre-Requirements

- Basic knowledge of C/C++ firmware development
- Familiarity with memory-mapped I/O and microcontroller peripherals
- VS Code (latest) with GitHub Copilot, or access to Claude Code
- Python 3.10+ for LangGraph exercises (Labs 006-007)
